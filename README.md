# Al Amin Robin Portfolio

A cinematic, high-performance architectural portfolio designed for the modern web. Built with React 19 and Vite, this application prioritizes visual excellence and technical precision.

## Core Pillar: Seamless Performance
- **0ms Atmosphere**: Critical CSS inlining ensures the foundational background and variables render instantly.
- **LazyMotion Architecture**: High-fidelity animations powered by Framer Motion, optimized to unblock the main thread.
- **Edge-Native Deployment**: Categorically optimized for Vercel with integrated Speed Insights and Analytics.
- **Asset Compression**: Pre-compressed Gzip assets for rapid global distribution.

## Technology Stack
- **Core**: React 19, JavaScript (ES6+), Vite
- **Styling**: Vanilla CSS (Custom Design System, Glassmorphism)
- **Animations**: Framer Motion, Lenis (Premium Smooth Scroll)
- **Infrastructure**: Vercel Analytics, Speed Insights, reCAPTCHA v3
- **Dev Tools**: ESLint, Rollup Bundle Visualizer

## Project Structure
- `/src/pages`: Cinematic page layouts (Home, About, Portfolio, Services, Contact)
- `/src/components`: Reusable UI components with micro-interactions
- `/public`: High-resolution architectural assets and media
- `vite.config.js`: Optimized build pipeline with compression and analytics
